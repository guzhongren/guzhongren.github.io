# “尿尿”


昨天回家我们吃完饭，西西就开始看她新一期巧的虎视频了。看视频的时候一会跟着视频说英语，一会拍手，一会又跳舞，一会又唱诗，玩的不亦乐乎。

看完了巧虎视频，她关掉了电视，我准备去主卧，看到他低着头走路，我好奇的说：“西西，你干嘛去？”

西西继续地址头，慢悠悠的说：“尿尿。”

听到西西说话，坐在沙发上的妈妈哈哈大笑起来，还嘲讽地说：“你娃长大了，不需要你了。”

西西还是低着头，找到了奶奶给她做的花色拖鞋，慢悠悠地穿上走进了卫生间，我们也没进去帮她脱裤子。

过了一会，我在卫生间门口看西西是不是已经把裤子都提好了。但我头一伸，西西看到之后就不提了，直接撒娇说：“爸爸，你帮我提一下，我不会提。”

我站在门外说：“屁股后面提一下，前面提一下就好了。”

西西还是一动不动地站着，然后又努力地提了一下屁股后面的裤子和前面的裤子，但提的地方不对，裤子始终提不起来，还时不时地发出声音：“哎哟，哎呦。”

最终，自己提裤子以失败告终，我不得不进去帮她提。

出来后，媳妇又嘲讽地说：“你娃已经不需要你了。”

我只能给她个眼神了。
