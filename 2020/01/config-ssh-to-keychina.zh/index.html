将生成的 ssh 私钥添加到 Mac 的 keychain 中， 如果是其他操作系统，可忽略此步骤

$ ssh-add -K .ssh/is_rsa

将登录信息配置到。ssh/config 中

$ touch ~/.ssh/config
$ vim ~/.ssh/config
# edit text
Host myvm
  Hostname ip
  User user

保存之后就可以使用如下命令快捷登录服务器了

$ ssh myvm
